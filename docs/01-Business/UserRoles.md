# User Roles & Hierarchy

1. **Super Administrator (Level 4)**: Global configurations, system logs, role assignment.
2. **Assistant Engineer / Admin (Level 3)**: Manages equipment, updates lab statuses, approves lab access, manages calibrations.
3. **Lecturer / Supervisor (Level 2)**: Approves student booking requests for academic tasks.
4. **Student / Researcher (Level 1)**: Initiates bookings, submits safety declarations, and files damage reports.
